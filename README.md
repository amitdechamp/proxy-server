# README #

Tiny and fast proxy server for axios based requests

All requests must be post request using axios format

# SETUP

Option 1
====

- git clone https://github.com/ayinlaaji/proxy-server.git
- cd proxy-server
- npm install
- npm install -g pm2
- npm start

===
